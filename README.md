# Tableau Desktop Skill Sprint

## Overview
This repository contains my Skill Sprint for MSBA 635, where I chose to learn **Tableau Desktop** using the 13‑day free trial. Tableau is a leading data visualization tool used for analytics, dashboards, and business storytelling. I used the Superstore dataset to build visualizations and create a simple dashboard.

## What This Repo Contains
- **implementation_plan.md** — My learning plan and timeline  
- **screenshot.png** — Screenshot of my completed Tableau dashboard  
- Additional practice files or screenshots may be added  

## Dataset Used
I used the **Superstore** dataset provided by Tableau.  
This dataset includes sales, profit, category, sub‑category, and customer information.

## What I Learned
- How to connect Tableau Desktop to a dataset  
- How to build bar charts and other basic visuals  
- How to use sorting, color, and formatting to improve clarity  
- How to create a dashboard combining multiple charts  
- How to present insights visually  

## Tools Used
- Tableau Desktop (13‑day trial)  
- Superstore sample dataset
